# Installation

## Via Docker
```console
foo@bar:~$ sh start.sh
```

## Via Terminal

### Front
```console
foo@bar:~$ cd backend/
foo@bar:~$ npm install
foo@bar:~$ npm run start
```

### Back
```console
foo@bar:~$ cd frontend/
foo@bar:~$ npm install
foo@bar:~$ npm run serve
```
