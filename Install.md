# Installation

## Via Terminal

### Front
```console
foo@bar:~$ cd backend/
foo@bar:~$ npm install
foo@bar:~$ npm run start
```

### Back
```console
foo@bar:~$ cd frontend/
foo@bar:~$ npm install
foo@bar:~$ npm run serve
```


## Via Docker (not supported for now)
```console
foo@bar:~$ docker-compose up
```
